Ressource : Présentation du module Bluetooth HC-06

La carte arduino.⚓

Le module Bluetooth doit être connectée à une carte programmable comme arduino ou picaxe.

Cette carte permet grâce au programme contenu dans la carte programmable d'envoyer et de recevoir une information numérique.

Cette carte a une portée de 10 mètres.

Pour une utilisation avec un smartphone, il est nécessaire d'appairer les deux appareils. Le code d'appairage par défaut est le code 1234.
